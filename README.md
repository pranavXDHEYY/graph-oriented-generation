## 📋 What is Graph-Oriented Generation?

Graph-Oriented Generation (GOG) is a user-friendly software designed to help you create and manage graph-based data. You can build relationship maps, diagrams, or network structures without any technical skills. This tool works on Windows and gives you a straightforward way to generate graphs visually.

You don't need to know programming or data science. Just follow simple steps to start working with your graphs.

## 🔍 Basic Features Overview

- **Create New Graph:** Start a blank canvas to build your graph.
- **Add Nodes:** Insert points to represent entities or data.
- **Connect Nodes:** Draw edges to show relationships.
- **Edit Labels:** Rename nodes and connections for clarity.
- **Visual Styles:** Change colors and shapes for easy understanding.
- **Save and Load:** Store your graphs for later use.
- **Export:** Save your graph as an image or PDF for sharing.

## 🎨 How to Create Your First Graph

1. Click "New Graph" from the main menu.
2. Use the "Add Node" button to place points on the canvas.
3. Select two nodes and click "Connect" to create relationships.
4. Double-click on nodes or edges to rename them.
5. Change colors using the toolbar to organize your graph visually.
6. Save your graph by clicking "File" > "Save As" and choosing a location on your computer.

The interface uses clear labels and icons to guide you.

## 💾 Saving and Exporting Files

After creating or updating a graph:

- Click "File" > "Save" to save progress.
- Use "File" > "Export" to save your graph as:
  - PNG image
  - JPEG image
  - PDF document

This allows you to share your work or include diagrams in other documents.

## 🔧 Customization Options

GOG offers settings to tailor your experience:

- Change background color of the canvas.
- Set default node and edge styles.
- Choose grid display on or off.
- Enable snap-to-grid for precise placement.

Access these in the "Settings" menu.
